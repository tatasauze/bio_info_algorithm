# Bio Info Algorithm

一個用於生物資訊學序列比對的 Needleman-Wunsch 演算法視覺化工具。

## 功能特色

- 互動式動態規劃矩陣網格視覺化
- 正向模式：查看每個單元格的計算邏輯
- 反向模式：追蹤最佳比對路徑
- 即時懸停互動
- 使用 D3.js 的現代化清潔 UI

## 使用方法

在網頁瀏覽器中開啟 `index.html` 即可查看視覺化效果。

## 演算法說明

Needleman-Wunsch 演算法用於全域序列比對，通過最大化評分函數來找到兩個序列之間的最佳比對。

### 評分標準
- 匹配：+1
- 不匹配：-1
- 間隙：-2
